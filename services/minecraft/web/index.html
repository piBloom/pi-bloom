Nazar Minecraft Server

Play Minecraft at mc.nazar.studio

A PaperMC survival server for the Nazar community. Vanilla clients can connect, and the recommended Prism Launcher setup below adds better performance, shaders, distant terrain, and voice chat support.

Server address

mc.nazar.studio

PaperMC survival

Online-mode server with a small set of server-side plugins.

Voice chat

Simple Voice Chat

Install the client mod below to join proximity voice chat.

Prism Launcher setup

  1. Install Prism Launcher.
  2. Open Prism Launcher and sign in with your Microsoft account.
  3. Create a new instance: Add Instance → Custom → Minecraft.
  4. Select the Minecraft version used by the server, then choose the Fabric loader.
  5. Open the instance folder, put the recommended .jar files in the mods folder, and start the game.
  6. In Multiplayer, add a server named Nazar with address mc.nazar.studio.
